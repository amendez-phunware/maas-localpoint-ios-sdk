Instance Methods

getBody

Returns the body of this message.

- (NSString *)getBody

Return Value

the body of this message

Discussion

The value returned by this method is usually HTML. This method should never return nil. The message impression event is sent along with this method got involked, so you should call it only when it's time to show user.

- (id<ILPID>)getID

Return Value
